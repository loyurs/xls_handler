# Python处理固定结构excel脚本

当前处理xls

提取当前文件夹下形如的xls文件，将其按照索引和值输出到一个汇总表，汇总表每行代表一张表的数据

| **idx1** | **1** | **idx2** | **2** |
| -------------- | ----------- | -------------- | ----------- |
| **idx3** | **3** | **idx4** | **4** |
| **idx5** | **5** |                |             |
|                |             | **idx6** | **6** |

如此类结构，第一列为索引，第二列为值的格式，将其统一输出到一个excel中，形如：

| **idx1** | idx2 | **idx3** | idx4 | idx5 | idx6 |
| -------------- | ---- | -------------- | ---- | ---- | ---- |
| 1              | 2    | 3              | 4    | 5    | 6    |

多张表形如


| **idx1** | idx2        | idx3 | idx4 | idx5 | idx6 |
| -------------- | ----------- | ---- | ---- | ---- | ---- |
| **1**    | **2** | 3    | 4    | 5    | 6    |
|                | **k** |      | k    |      | k    |
